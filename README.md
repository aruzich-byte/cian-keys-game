# Мини-игра (прототип Циан)

## Как запустить
- Открой `index.html` в браузере (Chrome/Safari)
- или через локальный сервер: `http://127.0.0.1:5173/`

## Механика
- Шарик управляется только влево/вправо.
- Сверху падают предметы:
  - ключи (4 вида) — good → `+1` балл
  - `вор`, `бабуля`, `невидимка` — bad → `-1` жизнь
- Всего `3` жизни (индикатор сверху).

## Картинки (3D/иконки) для падающих предметов
Положи PNG в папку:
`assets/game-items/`

Имена файлов должны совпадать с типами:
- `key1.png`
- `key2.png`
- `key3.png`
- `key4.png`
- `thief.png`
- `granny.png`
- `invis.png`

Если файла нет/не загрузится — останется CSS-графика (fallback).
